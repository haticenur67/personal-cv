# Personal CV Website

This is a modern responsive personal CV website created using HTML and CSS.

## 🚀 Features
- Fully responsive design (mobile, tablet, desktop)
- Modern glassmorphism UI design
- Animated background elements
- Interactive UI elements (hover effects on skills and projects)
- Clean and professional layout

## 🛠️ Technologies Used
- HTML5
- CSS3

## 🌐 Live Website
https://haticenur67.github.io/personal-cv/

## 📌 About
This project was built as a portfolio/CV assignment and also used as a personal showcase project for GitHub Pages deployment.
